# Orange Canvas Node Editor - Quick Start Guide

## 설치 및 실행

### 1. 패키지 설치
```bash
pip install -r requirements.txt
```

### 2. 애플리케이션 실행
```bash
python main.py
```

## 주요 기능 사용 방법

### 노드 추가 방법 (2가지)

#### 방법 1: 드래그 & 드롭
1. 좌측 "Node List" 패널에서 원하는 노드를 선택
2. 노드를 캔버스 영역으로 드래그
3. 마우스를 놓으면 노드가 추가됩니다

#### 방법 2: 우클릭 메뉴
1. 좌측 "Node List" 패널에서 노드에 우클릭
2. "추가" 메뉴 선택
3. 캔버스의 중앙에 노드가 추가됩니다

### 노드 조작

- **이동**: 노드를 드래그하여 이동
- **삭제**: 
  - 노드에 우클릭 → "삭제" 선택
  - 또는 노드를 선택 후 Delete 키 입력

### 파일 작업

- **새 프로젝트**: 파일 → 새로 만들기 (Ctrl+N)
- **저장**: 파일 → 저장 (Ctrl+S)
- **열기**: 파일 → 열기 (Ctrl+O)

### 편집 작업

- **되돌리기**: 편집 → 실행 취소 (Ctrl+Z)
- **재실행**: 편집 → 다시 실행 (Ctrl+Y)

### 로그 확인

- 하단의 "Log" 패널에서 모든 작업 이력 확인
- "로그 지우기" 버튼으로 로그 초기화

## 노드 분류

### Data 카테고리
- **Data Source**: 기본 데이터 소스
- **CSV File**: CSV 파일에서 데이터 로드
- **SQL Query**: SQL 쿼리 실행
- **Random Data**: 임의의 데이터 생성
- **Excel**: Excel 파일에서 데이터 로드

### Processing 카테고리
- **Filter**: 데이터 필터링
- **Aggregate**: 데이터 집계
- **Sort**: 데이터 정렬
- **Group By**: 데이터 그룹화
- **Join**: 여러 데이터 조인

### Visualization 카테고리
- **Table View**: 테이블로 데이터 표시
- **Chart**: 차트 생성
- **Scatter Plot**: 산점도 생성
- **Histogram**: 히스토그램 생성
- **Heatmap**: 히트맵 생성

## 팁과 요령

1. **조직적인 캔버스 유지**: 정기적으로 노드를 정렬하여 보기 좋은 레이아웃 유지
2. **로그 확인**: 문제 발생 시 하단 로그를 확인
3. **자주 저장**: 중요한 작업은 자주 저장
4. **Undo/Redo 활용**: 실수 시에는 Ctrl+Z로 되돌리기

## 문제 해결

### 노드가 추가되지 않는 경우
- 로그를 확인하여 에러 메시지 확인
- 드래그 앤 드롭 또는 우클릭 메뉴 방법 시도

### 파일을 열 수 없는 경우
- 파일이 올바르게 저장되었는지 확인
- .canvas 파일 형식인지 확인

### 애플리케이션이 응답하지 않는 경우
- 로그를 확인하여 문제 파악
- 애플리케이션 재시작 후 마지막 저장본 열기

## 향후 계획

- Node 간의 연결선 시각화
- Node 실행 및 데이터 흐름 시뮬레이션
- 플러그인 시스템 추가
- 더 많은 노드 타입 지원
