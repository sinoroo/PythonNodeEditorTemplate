# Orange Canvas - Node Editor

PySide6 기반의 노드 에디터 GUI 애플리케이션입니다. Orange3 프로그램을 참고하여 개발되었습니다.

## 특징

### 노드 시스템
- **3가지 카테고리**: Data (데이터), Processing (처리), Visualization (시각화)
- **각 카테고리 5개의 노드**: 총 15개의 예제 노드
- **입출력 포트**: 각 노드는 Input/Output 연결점을 가짐
- **노드 이미지와 이름**: 시각적 식별이 가능

### UI/UX 기능
- **Docking 패널**: 노드 목록과 로그 패널은 독립적으로 조정 가능
- **드래그 & 드롭**: 노드 리스트에서 캔버스로 드래그하여 노드 추가
- **컨텍스트 메뉴**: 노드 리스트에서 우클릭하면 "추가" 메뉴 표시
- **그리드 배경**: 캔버스에 그리드로 노드 정렬 용이

### 편집 기능
- **Undo/Redo**: 모든 작업 되돌리기 및 재실행
- **로그 시스템**: 하단 로그 패널에서 모든 작업 추적
- **저장/로드**: 캔버스 상태를 파일로 저장하고 로드

### 메뉴
- **파일**: 새로 만들기, 열기, 저장, 종료
- **편집**: 실행 취소, 다시 실행, 캔버스 지우기
- **보기**: 뷰 초기화
- **도움말**: 정보

## 노드 카테고리

### Data (데이터)
1. **Data Source** - 데이터 소스
2. **CSV File** - CSV 파일 입력
3. **SQL Query** - SQL 쿼리
4. **Random Data** - 랜덤 데이터 생성
5. **Excel** - Excel 파일 입력

### Processing (처리)
1. **Filter** - 데이터 필터
2. **Aggregate** - 데이터 집계
3. **Sort** - 데이터 정렬
4. **Group By** - 데이터 그룹화
5. **Join** - 데이터 조인

### Visualization (시각화)
1. **Table View** - 테이블 보기
2. **Chart** - 차트
3. **Scatter Plot** - 산점도
4. **Histogram** - 히스토그램
5. **Heatmap** - 히트맵

## 사용 방법

### 노드 추가
1. **왼쪽 패널의 Node List에서 노드 선택**
   - 카테고리를 확장하여 원하는 노드 선택
   
2. **두 가지 방법으로 노드 추가**
   - **드래그 & 드롭**: 노드를 드래그하여 캔버스에 놓기
   - **우클릭 메뉴**: 목록에서 우클릭하여 "추가" 선택

### 노드 연결
- 노드의 출력 포트와 다른 노드의 입력 포트를 연결할 수 있습니다
- (향후 버전에서 시각적 연결 기능 추가 예정)

### 파일 관리
- **저장**: 파일 → 저장 (Ctrl+S)
- **열기**: 파일 → 열기 (Ctrl+O)
- **새로 만들기**: 파일 → 새로 만들기 (Ctrl+N)

### 편집
- **실행 취소**: 편집 → 실행 취소 (Ctrl+Z)
- **다시 실행**: 편집 → 다시 실행 (Ctrl+Y)
- **캔버스 지우기**: 편집 → 캔버스 지우기

## 로그 확인
- 하단의 로그 패널에서 모든 작업의 로그를 확인할 수 있습니다
- "로그 지우기" 버튼으로 로그를 초기화할 수 있습니다
